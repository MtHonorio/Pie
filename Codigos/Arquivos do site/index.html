Arduino

Utilizamos a IDE do arduino para configurar a Wemos D1R2 e enviar e receber dados do firebase.

Firebase

Utilizamos o firebase para hospedar os dados, modificando os valores conforme fosse necessário.

Google Maps

Utilizamos o Google Maps para exibir o local das coordenadas que adquirimos do firebase.

Html, Css e Javascript

Utilizamos Html, Css e Javascript para exibir os dados obtidos do servidor de maneira mais fácil.

Equipamentos Principais

Utilizamos a Wemos D1R2 para comunicação com o servidor e o GY-NEO6MV2 para obter localização em tempo real.

000Webhost

Optamos por utilizar o 000Webhost para hospedar nosso site online, mostrando os dados que a Wemos D1R2 coleta.
